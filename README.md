# DevAgent — AI Coding Assistant

AI-ассистент для разработчиков с интеграцией GitHub API.

## Возможности
- 🧠 Coding Assistant — актуальные технологии 2025-2026
- 🐙 GitHub Agent — мониторинг репозиториев, issues, PR
- 🔍 Code Review — анализ кода на баги и безопасность
- 🏗️ Architect — помощь с архитектурой проекта

## Как использовать
1. Открой `dev-agent.html` в браузере
2. Вставь Anthropic API ключ
3. Опционально — GitHub Token для работы с репо

## Стек
- Vanilla JS + HTML/CSS
- Anthropic Claude API
- GitHub REST API
